1. Встановити Node.js версії v12.14.0 або вище
2. Встановити Mongodb
3. Конфігурація:
	Server: хост і порт на якому запускаться сервер
	Kontramarka:
		username, password: логін і пароль користувача контрамарки
		aliveTimeout: таймаут зберігання сесії контрамарки в нашій базі монго (в мілісекундах)
	Timers:
		timeoutSelectPlaces: таймаут відміченого місця (в мілісекундах)
		timeoutReservePlaces: таймаут заброньованого місця (коли перейшли до оплати) (в мілісекундах)
	useFinExpert: конфігурація мідлвара фінексперта, коли проводилось тестування для порівняння відповідей обох систем. "isUse": false - відключає мідлвар
		codeToEventIdFile: файл в якому записуються сеанси фінексперта відповідні до сеансів контрамарки
